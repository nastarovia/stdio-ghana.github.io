Why Object Oriented Programming
  • Common data and functions (inheritance)
  • Protect data (encapsulation)
Say I run a restaurant where I have several employees that have the same set of attributes that I care about; perform similar tasks; get paid certain amounts; etc. It is convenient to have template code that each instance (employee) can inherit quickly and I can reuse. Overview of OOP Terminology

Reference tutorialspoint

  • Class:A user-defined prototype for an object.
  • Inheritance:The creation of an instance of a class.
  • Instance:An individual object of a certain class.
  • Class:A user-defined prototype for an object that gives the object some attributes defined for any object of the class. The attributes are data members (class variables and instance variables) and methods. Methods are accessed via dot programming notation.
  • Inheritance:The creation of an instance of a class.
  • Instance:An individual object of a certain class. An object obj that belongs to a class Circle, for example, is an instance of the class Circle.
  • Instantiation:The creation of an instance of a class.
  • Method:A special kind of function that is defined in a class definition.
  • Object:A unique instance of a data structure that's defined by its class. An object comprises both data members (class variables and instance variables) and methods.

    Derived classes are written similar to their base classes

    class Employee:
    'Common base for all employees'
    empCount = 0

    def __init__(self, name, salary, isWorking):
        self.name = name
        self.salary = salary
        self.isWorking = isWorking
        Employee.empCount += 1

class Cook(Employee):
    'Common base for all cooks'

    Overriding methods can be useful

    class Vector:
   def __init__(self, a, b):
      self.a = a
      self.b = b

   def __str__(self):
      return 'Vector (%d, %d)' % (self.a, self.b)

   def __add__(self,other):
      return Vector(self.a + other.a, self.b + other.b)

v1 = Vector(2,10)
v2 = Vector(5,-2)
print v1 + v2 #This prints 'Vector(7,8)'
